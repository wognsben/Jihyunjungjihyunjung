# 워드프레스 이미지 설정 가이드

현재 Work 페이지에서 텍스트는 보이지만 이미지가 깨져(또는 회색 박스로) 보이는 이유는 **워드프레스 글에 '특성 이미지(Featured Image)'가 설정되지 않았기 때문**입니다.

Headless WordPress 방식에서는 본문에 이미지를 넣는 것과 별개로, 목록에서 보여줄 **대표 이미지**를 반드시 지정해야 API가 이를 가져올 수 있습니다.

## 해결 방법 (이미지 등록하기)

1. **워드프레스 관리자 페이지 접속**
   - URL: `https://wognsben97.mycafe24.com/wp-admin/`
   - 로그인하세요.

2. **글(Work) 편집 화면으로 이동**
   - 왼쪽 메뉴에서 **'Work'** (또는 '글') > **'모든 글'**을 클릭합니다.
   - 이미지가 안 나오는 글의 제목을 클릭하여 편집 화면으로 들어갑니다.

3. **특성 이미지(Featured Image) 설정**
   - 편집 화면 **오른쪽 사이드바**를 확인합니다. (보이지 않는다면 오른쪽 상단 설정 아이콘 ⚙️ 클릭)
   - 스크롤을 아래로 내려 **'특성 이미지(Featured Image)'** 항목을 찾습니다.
   - **'특성 이미지 설정(Set featured image)'**을 클릭하고 이미지를 업로드하거나 선택합니다.

4. **저장 및 업데이트**
   - 오른쪽 상단의 **'업데이트(Update)'** 버튼을 눌러 저장합니다.

5. **확인**
   - 이제 프론트엔드 사이트를 새로고침하면 이미지가 정상적으로 나타납니다.

---

## 참고 사항
- 코드는 이미지가 없을 경우 자동으로 "NO IMAGE"라는 문구와 함께 깔끔한 회색 박스를 보여주도록 업데이트되었습니다.
- 본문에 삽입된 이미지는 상세 페이지 갤러리에는 자동으로 추가되지만, 리스트(썸네일)에는 '특성 이미지'가 우선적으로 사용됩니다.
